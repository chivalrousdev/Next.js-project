# Thumbnail Slider Example

This is a "thumbnail slider" which allows a one slider to control another, larger slider. In this case, the lower slider is free scrolling while the other snaps. Click on any slide in the lower slider to see the effect. See [synchronized sliders](/docs/slider/setup#sync) for more information.
